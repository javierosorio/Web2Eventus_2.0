Web2Eventus Version 2.0
Release date: October, 2014


-------------------------------------------------------------------------------
1. GENERAL USAGE NOTES
-------------------------------------------------------------------------------
Web2Eventus extracts and reformats the information content of previously 
downloaded news reports to create a single document to be used as text corpus 
in Eventus ID. The program processes each news report by extracting the content, 
breaking it down by paragraphs, attaching a paragraph counter, formatting the 
information and storing all the text into a file readable in Eventus ID.


-------------------------------------------------------------------------------
2. INSTALLATION REQUIREMENTS
-------------------------------------------------------------------------------
Web2Eventus runs on Windows command line interface.

Web2Eventus also requires having Perl 5, or any later version.

Web2Eventus requires no specific installation procedure. All it needs is to 
have all the files indicated in the File Manifest contained in the same folder. 

Refer to the User Manual for operation instructions.

-------------------------------------------------------------------------------
3. FILE MANIFEST 
-------------------------------------------------------------------------------
The Web2Eventus download package contains the following files and folders:

- "web2eventus.pl" is the Web2Eventus v2.0 program file in Perl format.

- "Web2Eventus_2.0_Manual.pdf" is the User Manual.

- "corpus_DEMO.txt" is the sample text corpus generated by Web2Eventus.

- "web_DEMO" is a folder containing a collection of press releases in .html 
  format issued by the Mexican Army (Secertaria de la Defensa Nacional, SEDENA).
  These files are used for Web2Eventus to generate a sample corpus of text to
  be used by Eventus ID.

- "SEDENA_permission" is a folder containing the permission given by SEDENA for
  using and sharing the press releases. It also includes the required citation 
  of all the press releases used in the demonstration file.

- "GNU general public license.txt" is a file indicating the terms of use.
